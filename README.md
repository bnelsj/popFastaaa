# popFastaaa

popFastaaa are simple stand-alone programs for analysis of aligned Multi-FASTA format.

####Availible analyses:
- [X] Nucleotide diversity (PI)
- [ ] Theta
- [ ] Allele frequency


### installing

```
git clone --recursive https://github.com/zeeev/popFastaaa.git 
cd popFastaaa
make
```
