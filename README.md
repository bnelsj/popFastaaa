# popFastaaa

Simple stand-alone programs for analysis of aligned Multi-FASTA formated files.

####Availible analyses:
- [X] Nucleotide diversity (PI)
- [X] Alignment to SNP
- [ ] Theta
- [ ] Allele frequency


### Installing

```
git clone --recursive https://github.com/zeeev/popFastaaa.git 
cd popFastaaa
make
```
